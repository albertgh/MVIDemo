

## Overview

- SwiftUI + MVI

## Requirements

- Xcode 26+, iOS 18+, Swift 5.9+
- Network access for the production data source
- SPM dependency: `https://github.com/albertgh/ACEnhancement.git`

## Setup & Run

1. Clone the repository.
2. Open `xcodeproj` in Xcode.
3. If SPM packages are not resolved, go to `File > Packages > Resolve Package Versions`.
